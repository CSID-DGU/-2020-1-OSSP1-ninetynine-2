## 프로젝트 주제
- 얼굴 인식과 홍채인식의 기술이 상용화 되어 사진으로 인한 개인 정보의 유출 위험 존재.
- 대부분 얼굴을 스티커로 가리거나 모자이크 처리를 하지만 모자이크는 역 변활 가능.

- 따라서 사람의 얼굴을 자연스러운 가짜 얼굴로 대체하여 사진으로 인한 개인 정보 유출 보안.
